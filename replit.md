# Lists and Stacks Lab (C)

## Overview
A C programming lab exercise focused on implementing ArrayList (List) and Stack data structures. Students implement functions in `exercises.c`.

## Project Structure
- `exercises.c` — Student implementation file (the only file to be modified)
- `arraylist.h` / `arraylist.c` — List TDA implementation
- `stack.h` — Stack TDA header
- `test.c` — Test suite
- `test.sh` — Build and test runner script

## Running Tests
Open the Shell and run:
```bash
bash test.sh
```

## Workflow
- **Start application**: Runs `bash test.sh` — compiles `test.c` with gcc and executes the test suite.

## Language & Tools
- C (GCC 14.3)
- No frontend or backend web server
- No external dependencies
